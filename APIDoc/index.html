Introduction

PrysmisAI API

The PrysmisAI API gives you direct access to multiple AI models through a single OpenAI-compatible endpoint. No keys, no accounts, no quotas. Send messages, get responses.

The API is OpenAI-compatible. Any client that works with OpenAI's Chat Completions API will work here with a URL change.

Endpoint

There is one endpoint for all chat completions:

POST https://prysmisai.wtf/v1/chat/completions

You can also pass the model as a query parameter:

POST https://prysmisai.wtf/v1/chat/completions?model=gpt-5.2

Request Format

Send a JSON body with the following fields:

FieldTypeRequiredDescription
modelstringNoAI model ID. Defaults to gpt-5.2.
messagesarrayYesArray of message objects with role and content.
temperaturenumberNo0–2. Controls randomness. Default: 0.7.
max_tokensnumberNoMax tokens in response. Default: 4096.

Each message object:

FieldValuesDescription
rolesystem / user / assistantWho sent the message.
contentstring or arrayMessage text. Pass an array for vision (image + text).

Examples

Basic request (JavaScript)
const response = await fetch('https://prysmisai.wtf/v1/chat/completions', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    model: 'gpt-5.2',
    messages: [
      { role: 'system', content: 'You are a helpful assistant.' },
      { role: 'user', content: 'Write a Roblox health bar script.' }
    ],
    temperature: 0.7,
    max_tokens: 2048
  })
});

const data = await response.json();
console.log(data.choices[0].message.content);
Python
import requests

res = requests.post('https://prysmisai.wtf/v1/chat/completions', json={
    'model': 'claude-sonnet-4-5',
    'messages': [
        {'role': 'user', 'content': 'Create a Roblox shop system.'}
    ],
    'max_tokens': 4096
})

print(res.json()['choices'][0]['message']['content'])
cURL
curl -X POST https://prysmisai.wtf/v1/chat/completions \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-4o",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'

Response Format

Responses follow the OpenAI Chat Completions format:

{
  "id": "chatcmpl-...",
  "object": "chat.completion",
  "model": "gpt-5.2",
  "choices": [
    {
      "index": 0,
      "message": {
        "role": "assistant",
        "content": "The AI response text here..."
      },
      "finish_reason": "stop"
    }
  ],
  "usage": {
    "prompt_tokens": 24,
    "completion_tokens": 180,
    "total_tokens": 204
  }
}

Access the response text at: data.choices[0].message.content
